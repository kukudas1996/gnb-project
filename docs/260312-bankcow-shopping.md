# 뱅카우 한우쇼핑 서비스

> 뱅카우의 커머스 서비스. 뱅카우 투자 소 및 국내산 한우를 직접 구매할 수 있는 쇼핑 플랫폼.

---

## 1. 서비스 개요

- **서비스명**: 뱅카우 한우쇼핑
- **운영사**: 스탁키퍼 (Stockeeper)
- **플랫폼**: 웹 (bankcow.co.kr/shop) / 앱은 웹뷰로 GNB 업데이트 후 진입 예정
- **판매 상품**: 뱅카우 투자 소에서 나온 한우 + 일반 한우 상품
- **가공센터**: 경기도 곤지암 뱅카우 스마트팩토리 (고기설계소 1센터), HACCP 인증 자체 운영
- **배송 방식**: 신선 택배 (택배사 가변)

---

## 2. 투자 서비스와의 연결

| 기능 | 설명 |
|------|------|
| 포인트 사용 | 투자 리워드로 지급된 포인트를 쇼핑 결제에 사용 (1P = 1원) |
| 내가 키운 소 구매 | 내가 투자한 소가 출하된 경우 해당 한우 직접 구매 가능 |
| 투자자 전용 할인 | 투자자 대상 별도 할인 혜택 제공 |
| 멤버십 (예정) | 멤버십 등급에 따라 포인트 및 쿠폰 추가 지급 예정 |

---

## 3. 상품 구조

- **등급 체계**: 1++(9)등급 / 1++등급 / 1+등급 / 1등급 / 2등급 / 기타
- **상품 유형**: 구이용, 국·찌개용, 불고기·샤브샤브, 스테이크 등 조리용도별 구성
- **상품 단위**: 팩 단위 판매 (2팩, 3팩 등 묶음 구성)
- **포장 방식**: 특수진공스킨포장 적용 → 냉장 상태 최대 2주 보관 가능
- **보관 형태**: 냉장육 (-2~5°C) / 냉동육 (-18°C 이하)

---

| 결제 수단 | 설명 |

| 토스페이 | 외부 간편결제 |

| 쿠폰 | 사용 가능한 쿠폰 선택 적용 (쿠폰 간 중복 사용 불가) |

## 5. 배송 정책

| 항목 | 내용 |
|------|------|

| 예약 상품 | 생산 당일 출고 예정 |

---

## 6. 포인트 정책

- **단위**: 1P = 1원
- **사용처**: 한우쇼핑 결제 시 사용
- **지급 조건**
  - 투자 이벤트 시 투자량에 따라 지급
  - 한우쇼핑 구매 촉진 목적으로 별도 지급 (이벤트성)
  - 뱅카우 계좌로 50,000원 이상 결제 시 3,000P 적립
  - 멤버십 등급에 따라 추가 지급 예정
- **만료 정책**: 포인트별 소멸 예정일 별도 표시
- **취소 시 처리**: 주문 취소 시 사용 포인트는 환불되지 않으며, 취소일로부터 7일간 사용 가능한 포인트로 재발급 → 7일 이내 미사용 시 자동 소멸

---

## 7. 쿠폰 정책

- 쿠폰 간 중복 사용 불가
- 할인 금액이 상품 금액 초과 시 차액 소멸
- 금액 할인 쿠폰은 상품 금액에만 적용 (배송비 제외)
- 무료배송 쿠폰은 배송비에만 적용
- 유효기간 내에만 사용 가능, 환급 불가
- 주문 취소 시 쿠폰 재지급, 유효기간은 최초 발급 기준 유지

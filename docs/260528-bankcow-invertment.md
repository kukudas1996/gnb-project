# 뱅카우 한우투자 서비스

> 뱅카우의 핵심 서비스. 한우를 증권 구조로 공동 투자하는 핀테크 플랫폼.

---

## 1. 서비스 개요

- **서비스명**: 뱅카우 (Bancow)
- **슬로건**: 송아지 키우며 자산을 늘려요
- **운영사**: 스탁키퍼 (Stockeeper)
- **포지셔닝**: 국내 최초 금융당국 인정 제도권 가축투자 금융상품
- **투자 구조**: 한우 공동 소유 증권 방식 (한우 소유권 지분 투자)
- **투자 기간**: 약 2년 (송아지 6개월령 매입 → 평균 30개월령 출하)
- **투자 단위**: 1C = 20,000원 (뱅카우 고유 단위)
- **최소 투자**: 1C = 20,000원
- **최대 투자**: 상품당 1,500C (30,000,000원)
- **체결 C**: 실제로 투자가 완료된 C 수량. 멤버십 등급 산정 기준으로 사용됨 (누적 체결 C 기준, 매월 1일 재산정)
- **결제 구조**: NH농협 가상계좌 발급 → 실물계좌 연결 → 입출금
- **플랫폼**: iOS / Android 앱

---

## 2. 비즈니스 구조

```
투자자 (소비자)              농가 (생산자)
    ↓ 송아지 구매 투자            ↓ 사육비 부담 + 사육
         ↘                   ↙
           공동 자산 (한우)
                ↓ 약 2년 후 경매 출하
           수익 공유 (투자 비율대로)
```

- 송아지 구매가 약 400만원 + 사육비 약 400만원 구조
- 투자자는 송아지 구매 단계부터 직접 참여
- 출하 후 수익을 투자금액 비율대로 분배
- 스탁키퍼는 뱅카우 외 '고기설계소', '솔직한우'를 통해 사육→경매→생산→유통 밸류체인 전 과정 직접 운영

## 3. 수익 / 리스크 구조

| 항목 | 내용 |
|------|------|
| 예상 수익률 | 마리당 평균 19.7% (2년 기준, 연환산 약 10%) |
| 플랫폼 이용료 | 200원/건 (VAT 포함) |
| 청약취소 수수료 | 1,000원/건 (VAT 포함) |
| 세금 | 배당소득세 14% + 지방세 1.4% = 15.4% 원천징수 (2025.7.1 지급분부터 / 이전은 기타소득 과세) |
| 리스크 | 경매 시점 한우 육질등급에 따라 개인별 수익률 차이 발생. 원금손실 0~100% 가능 |
| 보험 | 가축재해보험 가입 → 제1종 법정 가축전염병 발생 시 원금 100% 보상 |
| 예금자보호 | 예금자보호법에 따라 예금보험공사가 보호하지 않음 |
| 과거 실적 | 과거 운용실적이 미래 수익률을 보장하지 않음 |
| 양도·거래 | 양도 및 개인 간 거래 불가 |
| 투자 권고 | 일반투자자 여유자산의 10% 초과 투자 자제 권고 |
| 분산투자 권장 | 한 마리 집중 투자보다 여러 상품 분산 투자 권장 |

**한우 육질등급과 수익의 관계**
- 출하 시 한우 육질등급(1++, 1+, 1, 2, 3)에 따라 경매가 결정
- 등급이 높을수록 경매가가 높아 투자 수익 증가
- 상품 선택 시 유전자 수 / 유전력 데이터로 품질 예측 가능

**시장 데이터**
- 한우 시장 연평균 5.6% 성장
- 등심 소비자 가격 10년간 65.4% 증가
- 10년간 한우 경매 마리당 평균 수익률 19.7%

## 4. 상품 구조

- **현재 모집 중**: 7-1호
- **이전 상품 상태**: 정산 완료 / 사육 중 혼재
- **상품 번호 체계**: 1호 ~ 7-1호 (서브 넘버 포함, 예: 6-2호)
- **상품별 차이**: 투자 금액 범위, 기간, 수익률 조건이 상품마다 다름 (큰 틀은 유사)
- **상품 구성 방식**: 여러 마리 송아지를 하나의 상품으로 구성 → 변동성 리스크 분산
- **회사 선매입**: 상품 모집 시작 전 스탁키퍼가 전체 물량의 10%를 선매입
- **모집 마감**: 상품별 마감일 기준으로 마감. 100% 미달 시 마감 처리 후 전액 환불

### 상품 상태값

| 상태 | 설명 |
|------|------|
| 모집 중 | 현재 투자 가능한 상품 |
| 사육 중 | 투자 완료 후 농가 사육 진행 중 |
| 만기 | 출하 완료, 정산 진행 중. 소 시세·사육 상황에 따라 출하 시점이 유동적일 수 있음 |
| 정산 완료 | 수익 지급 완료 |

## 5. 계좌 구조

```
실물계좌 (사용자 기존 은행 계좌)
    ↕ 연결
NH농협 가상계좌 (뱅카우 전용)
    ↕ 입출금
투자 상품
```

- 가상계좌는 회원가입 시 자동 발급
- 입금/출금 모두 가상계좌를 통해서만 가능
- 만기 시 수익금이 가상계좌로 입금 → 사용자가 실물계좌로 직접 출금
- 토스뱅크 포함 주요 은행 연결 가능

**취소 / 환불**
- 투자 신청 단계(체결 전)에만 취소 가능, 취소 수수료 1,000원/건 부과
- 체결 후 취소 불가
- 부분 취소 불가 (전액 취소만 가능)
- 환불은 투자 신청한 회원 명의 가상계좌로만 처리

## 9. 투자 방법 / 환불 방식

### 투자자 유의문구 (앱 내 표시)
- 청약 모집률 100% 달성해도 조기마감되지 않음
- 분배방법: 일반 배정
- 플랫폼 이용료: 200원/건 (VAT 포함)
- 청약취소 수수료: 1,000원/건 (VAT 포함)

### 모집 방법
- 일반 공모 방식 (균등 · 비례 · 추첨)
- 공동사업 운영자(스탁키퍼)에게 전체 투자계약증권의 10% 우선 배정, 나머지 90%는 일반 청약자 배정
- 총 청약 수량이 공모 수량 초과 시: 90% 배정분 중 10%를 균등 배분 후 나머지 80%는 청약 수량 비례 분배
- 위 과정 후 잔여 물량은 공동사업 운영자에게 추가 배정 가능

### 일반 배정 방식 (배정 산식)

균등 배정
- C 발행량의 10% ÷ 투자 신청 고객 수
- 소수점 0.5 이하 삭제, 0.6 이상 올림 처리 후 선착순 분배

비례 배정
- (고객 구매 C 수량 ÷ 전체 고객 구매 C 수량) × C 발행량의 80%
- 비율이 같은 경우 소수점 0.5 이하 삭제, 0.6 이상 올림 처리 후 선착순 분배

기타
- 추가 투자 신청 시 최초 투자 신청 시간으로 합산
- 투자 마감 후 최종 경쟁률에 따라 투자 신청금이 높은 순으로 C 분배 → 신청 C 수량과 체결 C 수량이 다르거나 투자 실패할 수 있음

### 권리 구조
- 권리 유형: 공유 지분권
- 투자자는 보유 증권 C수에 비례하는 비율로 공동사업 손익을 지급받을 권리 보유
- [제19조 사업손익] 투자자는 취득한 증권 C수에 비례하는 비율에 따라 이익과 손실을 지급받을 권리 있음
- [제29조 투자자총회와 의결권] 확정된 투자자명부에 기재된 투자자들이 투자자총회 구성 / 의결권은 증권 1C마다 1개

### 수익 구조
- 수익 발생 방식: 송아지 취득 → 사육 → 출하 → 경매
- 뱅카우는 투자계약증권 모집가액의 재원으로 기초자산(한우)의 취득·사육·매각을 진행
- 투자자는 한우 경매를 통해 발생한 손익을 취득한 투자계약증권의 보유 지분 비율에 따라 귀속받음

### 투자 방식
- 동일 상품 추가 투자 시 [추가 투자하기] 기능 사용 → 최초 C 구매 시간 기준으로 기존 C에 추가 C 합산
- 환불은 투자 신청한 회원 명의 뱅카우 전용계좌로만 처리
- 투자자 1인당 1개 상품 당 최대 1,500C
- 원금손실(0~100%) 가능 (송아지 폐사 및 한우 시장 가격 변동)
- 양도 및 개인 간 거래 불가
- 투자 전 상품설명서 및 약관 필독 안내

### 환불 방식
- 취소 동의 즉시 플랫폼 이용료 + 취소 수수료 제외한 투자금 즉시 환불
- 부분 취소 불가 (전액 취소만 가능)
- 환불은 투자 신청한 회원 명의 뱅카우 전용계좌로만 처리

## 10. 투자자 유의사항 (법적 고지)

> 출처: 가축투자계약증권 제8-1·8-2호 증권신고서 (2026.03.26 제출, ㈜스탁키퍼)

### 상품 성격
- 투자자가 민법상 공동소유권을 갖는 **투자계약증권** (주식·채권과 달리 비정형적 구조)
- 예금자보호법 적용 없음 — 원금 0~100% 손실 가능
- 만기까지 중도해지 불가, 양도 및 개인 간 거래 불가

### 주요 위험 요소

사업위험
- 경기 변동에 따른 한우 수요 감소 위험
- 사료비 변동 (전체 생산비의 40~50% 차지, 글로벌 곡물가·환율 영향)
- 전염병(구제역 등) 발생 시 폐사·살처분 위험
- 한우 경매가 변동에 따른 수익성 악화
- 공동사업 참여자(농가) 사육 역량에 따른 수익 차이

회사위험
- 수익성: 2025년 매출 226억, 영업이익 3억 (흑자 전환), 부채비율 118.6%
- 공동사업 외부 감시·감독 기관 없음
- 공동사업자 도산 시 청산 업무 대행 체계 구축으로 대응

기타위험
- 환금성 제약: 한국거래소 미상장, 활성 유통시장 없음
- 원금 미보장: 실적배당형 상품
- 공모가 산정방식의 한계 존재 (발행인 이익 과다 산정 가능성)
